## About

Survev Crosshair is a modern crosshair generator made for the game Survev.io.

This project was created as a complete remake and improvement of alimg's old crosshair generator. While the original tool was a great idea for its time, it had very limited customization, only supported a single crosshair style, and had been left outdated and unmaintained for years.

Survev Crosshair aims to be the definitive replacement with a cleaner interface, significantly more customization, better visuals, and modern export and sharing features.

Built entirely with JavaScript, the generator allows you to create highly customizable and high-quality crosshairs with real-time editing and advanced tweaking options.

---

## Features

- Multiple crosshair types
  - Dot
  - Chevron
  - Classic
  - Custom styles
  - And more

- Extensive customization
  - Size
  - Thickness
  - Gap
  - Rotation
  - Opacity
  - Outline
  - Colors
  - Shadows
  - Scaling
  - Advanced positioning

- Modern UI
  - Smooth interface
  - Fast real-time updates
  - Minimal and responsive design

- Export options
  - Export as bookmarklet
  - Export as userscript

- Crosshair sharing
  - Generate share strings
  - Load shared crosshairs directly into the editor

- Community gallery
  - Browse crosshairs made by other players
  - Instantly use gallery presets
  - Submit your own creations

## Gallery

The Gallery is a built-in section of the website where users can explore crosshairs created by the community.

You can:
- Preview crosshairs
- Import them instantly
- Discover new styles
- Share your own setups

To submit a crosshair to the gallery:

1. Create your crosshair
2. Export or save the share string
3. Send it to me on Discord

Discord: `piesimp`

## Why This Exists

The original crosshair generator by Alimg had not been updated for many years and lacked the flexibility many players wanted.

Survev Crosshair was made to modernize the entire experience:
- Better visuals
- Better customization
- Better sharing
- Better exports
- Better usability

Everything was redesigned from the ground up to feel smooth, modern, and powerful while still being easy to use.
